# YOLO-DCARF
This is github repository for my proposed model used in research paper
DATASET LINK:-
// HIT-UAV Dataset: https://www.kaggle.com/datasets/pandrii000/hituav-a-highaltitude-infrared-thermal-dataset
// IndraEye Dataset: https://www.kaggle.com/datasets/paritoshkaggle1/myindraeye

Change dataset path when you run a model as per environment
The environment setting which i used for implimentation of my model is discussed in research paper ensure to keep same environment for similar result.
myYOLO-DCARFcode.py contains the main code of YOLO-DCARF Model. 
visualization_code.py file contains code used to generate graphs and visualization of result.
yolo-dcarf-1.ipynb & modified-indraeye.ipynb is a code and output of model which i got when run on kaggle.
